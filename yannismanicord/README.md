# Portfolio (style pierre.software-like) — React + Vite + Tailwind + Docker

## Lancer en local (dev)
```bash
npm install
npm run dev
```

## Lancer en production (Docker)
```bash
docker build -t portfolio .
docker run -p 3000:80 portfolio
```
Ouvre http://localhost:3000

## Raccourcis clavier
H = Explore · S = Stack · E = Experiences · P = Projects · A = About · T = Terminal  
? = Surprise · U = ShukaScript · Esc = fermer le toast

## Personnaliser
Modifie `src/data.js` (nom, liens, stack, projets, expériences).
